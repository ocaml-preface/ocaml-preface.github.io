Using the minimal definition

Build an Preface_specs.STATE over an Inner Preface_specs.MONAD.

module Over_monad : functor (M : Preface_specs.MONAD) -> functor (State : Preface_specs.Types.T0) -> Preface_specs.STATE with type state = State.t and type 'a monad = 'a M.t

Improving API

If there are complementary implementations to Preface_specs.MONAD attached to the Inner Preface_specs.MONAD type, it is possible to promote the Preface_specs.STATE API with these complementary implementations

Functor

If the Inner Preface_specs.MONAD is also a Preface_specs.FUNCTOR, the State monad is also a Preface_specs.FUNCTOR.

module Functor : functor (F : Preface_specs.FUNCTOR) -> functor (State : Preface_specs.Types.T0) -> Preface_specs.FUNCTOR with type 'a t = State.t -> ('a * State.t) F.t

Applicative

If the Inner Preface_specs.MONAD is also an Preface_specs.MONAD, the State monad is also an Preface_specs.APPLICATIVE.

module Applicative : functor (M : Preface_specs.MONAD) -> functor (State : Preface_specs.Types.T0) -> Preface_specs.APPLICATIVE with type 'a t = State.t -> ('a * State.t) M.t

Alternative

If the Inner Preface_specs.MONAD is also an Preface_specs.MOAND_PLUS, the State monad is also an Preface_specs.ALTERNATIVE.

module Alternative : functor (M : Preface_specs.MONAD_PLUS) -> functor (State : Preface_specs.Types.T0) -> Preface_specs.ALTERNATIVE with type 'a t = State.t -> ('a * State.t) M.t

Monad

If the Inner Preface_specs.MONAD is also a Preface_specs.MONAD, the State monad is also a Preface_specs.MONAD (this is this functor which is used in the full API definition).

module Monad : functor (M : Preface_specs.MONAD) -> functor (State : Preface_specs.Types.T0) -> Preface_specs.MONAD with type 'a t = State.t -> ('a * State.t) M.t
