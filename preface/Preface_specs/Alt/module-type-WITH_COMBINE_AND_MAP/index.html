Module type Alt.WITH_COMBINE_AND_MAP

The minimum definition of an Alt. It is by using the combinators of this module that the other combinators will be derived.

type 'a t

A type 'a t held by the Alt.

val combine : 'a t -> 'a t -> 'a t

Combine two values of t into one.

val map : ('a -> 'b) -> 'a t -> 'b t

Mapping over t from 'a to 'b.
