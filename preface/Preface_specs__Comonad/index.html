Laws

  • extend extract = id
  • (extend %> extract) f = f
  • extend g %> extend f = extend (extend g %> f)
  • f =>= extract = f
  • extract =>= f = f
  • (f =>= g) =>= h = f =>= (g =>= h)
  • extract <% duplicate = id
  • fmap extract <% duplicate = id
  • duplicate %> duplicate = fmap duplicate <% duplicate
  • extend f = fmap f <% duplicate
  • duplicate = extend id
  • fmap f = extend (f <% extract)
